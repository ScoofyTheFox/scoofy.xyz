Route /assets/

Media folders and setup notes for the site.

Use this page as the quick reference for what goes where when you keep expanding the site.

Folders

  • `assets/images/` for art, renders, logos, drawings, and memes
  • `assets/gifs/` for Scoofyx gifverse loops
  • `assets/music/` for looping audio tracks and `tracks.json`
  • `assets/live/` for the live tab JSON and live images
  • `assets/settings/` for shared site settings

Music

Hidden background music setup

Drop an mp3 into `assets/music/` and add it to `tracks.json`. The site now plays music in the background with automatic track changes, fades, and a subtle echo effect, without showing a visible player box.

Live tab

GitHub-editable update feed

Edit `assets/live/live.json` to change the live page whenever you want. Add images in `assets/live/images/` and point to them from the JSON file.
