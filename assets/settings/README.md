# Site Settings

This folder is the shared settings layer for the site.

## Main file

- `site.json`

Use it for:

- music behavior
- live page source path
- page settings file map

## Page settings

Each page now has its own JSON file in:

- `assets/settings/pages/`

Current files:

- `home.json`
- `docs.json`
- `fursona.json`
- `gallery.json`
- `live.json`
- `furries.json`
- `links.json`
- `assets.json`

These files control the top editable copy for each page, like:

- page title
- meta description
- intro heading text
- intro paragraph text
- banner/hero labels on some pages
