THE MANUAL
OVERVIEW

HOW THIS SITE WORKS

Pure HTML + CSS + JS. No frameworks, no build tools, no node_modules folder of doom. Just files. Every page links to style.css for shared styles.

SETUP

DEPLOYING UPDATES

1. edit the .html file on GitHub
2. click "commit changes"
3. wait ~60 seconds
4. scoofy.xyz is live ✓
PAGES

ADDING NEW PAGES

Create a new .html file in your repo. Copy the nav from any existing page. Add your content. Then add a link to it in every page's nav.

newpage.html → scoofy.xyz/newpage
newpage.html or newpage/index.html
GAME

VOID RUNNER ENGINE

The game uses Three.js for 3D rendering. The leaderboard is stored in the browser's localStorage — every player who plays on the same device shares scores. To reset, open DevTools → Application → localStorage → delete scoofy_lb_v2.

CUSTOMIZE

CHANGING COLORS & STYLE

All colors live in style.css under :root. Change --cyan to any hex color and the whole site updates. Fonts are loaded from Google Fonts — change the import URL at the top to swap them out.
